# НАЗВА ГРИ

Похмурий 2D-екшен про журналіста, який суне носа куди не слід. Розслідування, бійки, діалоги та атмосфера нуару.

## Як запустити

Тобі потрібен лише Python.

1. Завантаж цей проект.
2. Відкрий папку в терміналі.
3. Введи ці дві команди:

```bash
pip install pygame
python src/main.py
```

Все. Гра працює.

## Управління

* WASD — Рух
* E — Взаємодія (говорити, зберігатися)
* ЛКМ — Атака
* Q — Постріл (парирування)
* Пробіл — Ривок
* F — Зміна режиму зброї
